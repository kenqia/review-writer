# Review Writer 前端使用说明

## 启动

```bash
python3 <review_root>/view/serve_review_dashboard.py --review-root <review_root> --host 127.0.0.1 --port 8765
```

浏览器打开：

```text
http://127.0.0.1:8765
```

## 页面

- `Library`：论文入库与 metadata 审核，查看 PDF、MinerU Markdown 和结构化元信息。
- `Discovery`：主题检索结果审核，删除不相关关键词和论文，确认后生成 `selected_discovery_results.json`。
- `Matrix`：查看文献矩阵、逐篇阅读总结、大纲候选和矩阵生成报告。
- `Blueprint`：查看章节任务书，检查每章写作目标、分配文献、段落计划和图表需求。
- `Sections`：查看分章节草稿、章节任务 JSON、章节写作报告和候选图列表。
- `Figures`：检查候选图是否已定位源图、是否完成统一风格重绘，以及是否仍需人工确认。
- `Draft`：查看和编辑合并润色后的初稿，查看 merge report 和 remaining issues。
- `Final`：查看终稿、全文审查报告和 release report，作为发布前最后检查入口。

## 人工检查顺序

```text
Library -> Discovery -> Matrix -> Blueprint -> Sections -> Figures -> Draft -> Final
```

写作前准备只需要使用 `Library`。正式综述任务从 `Discovery` 开始，后续页面都基于同一个 `review-projects/<project_id>` 展示对应阶段产物。
