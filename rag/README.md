# RAG

This directory contains Retrieval-Augmented Generation planning assets for review-writer.

Current status:

- Phase 6a only provides no-upload preflight checks.
- No file upload, knowledge-base creation, or provider API call is performed by default.
- Real RAG pilots must be explicitly authorized by the user and must use a clean, human-reviewed dataset.

Start with:

```bash
make bailian-rag-preflight-check
```
